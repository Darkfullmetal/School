Project Proposal: FFXI Gear Swap Management

Project Concept | Site Map | Wireframe | Project Resources

1. PROJECT CONCEPT

Website Type & Objectives

This project involves the creation of a dynamic and user-friendly website designed to help FFXI players manage their character jobs, gear swaps, and wishlists. The main objectives are to:

  1. Create a platform for users to track and customize gear swaps for each job.
  2. Allow users to manage their wishlist of desired items (up to 100 slots).
  3. Provide quick access to FFXI resources and a discussion page.

Intended Audience

The intended audience is Final Fantasy XI players who want to organize their jobs, gear, and item goals efficiently.

Key Features

  • Home page with FFXI updates/tips, sign in/register, Hot topic list, and quick links (Character Page, Wishlist, discussion page, and Tips/links pages)
  • Sign in/register page with required fields: Username, Password, Email, Character Name
  • Character page showing jobs, levels, mission status, and assault checklist
  • Job pages, as sub pages to the Character page, with up to 5 customizable gear swap tables and wishlist, only for that job, access
  • Wishlist page, for the charchter, with up to 100 customizable slots

2. SITE MAP

Website Content Organization

The website is organized around the user’s FFXI character, with content grouped by jobs, gear swaps, wishlists, and helpful resources.

Sub-topics & Pages

  • Home/Overview
  • Sign In/Register
  • Character Page (job details, mission status, assault checklist)
  • Job Pages (up to 22, one per job, each with 5 gear swap tables and wishlist link)
  • Wishlist
  • Discussion Page
  • Tips/Links (BGwiki, FFXIAH, Official Site)

Page Layout & Navigation

Navigation will use a top menu for Home, Character, Wishlist, Discussion, and Tips/Links. Each job page is accessible from the Character page. Quick links to external sites open in a new window. Example - FFXI Wiki.

4. PROJECT RESOURCES

Helpful Resource

One valuable resource is the W3Schools Tabs Tutorial. This guide will help implement tabbed navigation for gear swap tables on each job page, making it easier for users to switch between different gear sets.

How It Will Be Used

  • To create interactive, user-friendly gear swap sections for each job
  • To improve the overall usability and organization of the job pages

Additional Project-Specific Resources

  • DataTables + Bootstrap Integration – For creating interactive, searchable, and sortable tables for gear swaps, including support for images and custom columns.
  • Bootstrap Images Documentation – To easily style and display gear icons within tables for a visually appealing layout.
  • Flask File Uploads – For allowing users to upload custom gear images or icons to personalize their gear swap tables.
  • Jinja2 For Loops – To dynamically generate table rows for each gear item in a user's gear swap set.
  • MDN <img> Tag Reference – For best practices on displaying and optimizing images in HTML tables.
